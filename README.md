# 🚀 Modern K8s Traffic Management: Gateway API & Envoy Gateway

এই প্রোজেক্টে আমি কুবারনেটেসের পরবর্তী প্রজন্মের নেটওয়ার্কিং স্ট্যান্ডার্ড **Gateway API** এবং **Envoy Gateway** ব্যবহার করে একটি মাইক্রোসার্ভিস রাউটিং সিস্টেম তৈরি করেছি। এটি প্রথাগত **Ingress**-এর একটি আধুনিক, ফ্লেক্সিবল এবং শক্তিশালী বিকল্প।

---

## 🎯 প্রোজেক্টের উদ্দেশ্য ও লক্ষ্য (Purpose & Goals)

বর্তমানে কুবারনেটেস নেটওয়ার্কিং লেয়ারে Ingress-এর সীমাবদ্ধতা দূর করতে **Gateway API** ব্যবহৃত হচ্ছে। এই প্রোজেক্টের মাধ্যমে নিচের বিষয়গুলো ইমপ্লিমেন্ট করা হয়েছে:

- **আধুনিক ট্রাফিক ম্যানেজমেন্ট**  
  প্রাচীন Ingress-এর বদলে আধুনিক `Gateway` এবং `HTTPRoute` রিসোর্স ব্যবহার করা হয়েছে।

- **স্মার্ট রাউটিং**  
  Path-based routing ব্যবহার করে ট্রাফিক বিভিন্ন মাইক্রোসার্ভিসে ডাইভার্ট করা হয়েছে।

- **ইন্ডাস্ট্রি স্ট্যান্ডার্ড আর্কিটেকচার**  
  Infrastructure routing এবং Application routing আলাদা করা হয়েছে (Separation of Concerns)।

- **লোকাল ক্লাউড ল্যাব**  
  `Kind` এবং `MetalLB` ব্যবহার করে লোকাল মেশিনেই LoadBalancer IP সেটআপ করা হয়েছে।

---

## 🏗️ Architecture

এই প্রোজেক্টে একটি **single entry point (Gateway)** ব্যবহার করে path-based routing করা হয়েছে।
